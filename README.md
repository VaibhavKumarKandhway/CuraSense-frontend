# CuraSense Dashboard (UI clone)

Folder: `d:/CuraSense-frontend`

To run locally (Windows PowerShell):

1. Open PowerShell and cd to the folder:

   cd 'd:\CuraSense-frontend'

2. Install dependencies:

   npm install

3. Start the dev server:

   npm start
